# 중복 제거하기
## 문제 설명
정수 배열 `nums`가 주어졌을 때, 중복된 숫자를 제거하고 남은 유일한 숫자들의 개수를 return 하도록 solution 함수를 완성해주세요.

배열의 순서는 유지하지 않아도 됩니다.

---

## 제한 사항

- `nums`의 길이는 0 이상 1,000 이하입니다.
- `nums`의 각 요소는 -1,000 이상 1,000 이하의 정수입니다.

---

## 입출력 예

| nums                  | result |
| --------------------- | ------ |
| [1, 2, 2, 3, 3, 3]    | 3      |
| [4, 5, 4, 5, 4, 5]    | 2      |
| [7, 7, 7, 7]          | 1      |
| [1, 2, 3, 4, 5]       | 5      |
| []                    | 0      |

---

## 입출력 설명
- 첫 번째 예시에서 [1, 2, 2, 3, 3, 3]은 유일한 숫자가 1, 2, 3으로 3개입니다.
- 두 번째 예시에서 [4, 5, 4, 5, 4, 5]는 유일한 숫자가 4, 5로 2개입니다.
- 세 번째 예시에서 [7, 7, 7, 7]은 유일한 숫자가 7로 1개입니다.
- 네 번째 예시에서 [1, 2, 3, 4, 5]는 모두 다른 숫자로 5개입니다.
- 다섯 번째 예시에서 빈 배열은 유일한 숫자가 0개입니다.
