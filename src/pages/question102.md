# 가장 많이 등장하는 문자
## 문제 설명
문자열 `s`가 주어졌을 때, 가장 많이 등장하는 문자를 return 하도록 solution 함수를 완성해주세요.

만약 여러 문자가 같은 횟수로 가장 많이 등장한다면, 알파벳 순서가 빠른 문자를 return 해주세요.

---

## 제한 사항

- `s`의 길이는 1 이상 1,000 이하입니다.
- `s`는 알파벳 소문자로만 이루어져 있습니다.

---

## 입출력 예

| s              | result |
| -------------- | ------ |
| "hello"        | "l"    |
| "aabbcc"       | "a"    |
| "programming"  | "g"    |
| "aaa"          | "a"    |
| "abcde"        | "a"    |

---

## 입출력 설명
- 첫 번째 예시에서 "hello"는 'l'이 2번으로 가장 많이 등장합니다.
- 두 번째 예시에서 "aabbcc"는 'a', 'b', 'c'가 모두 2번씩 등장하므로 알파벳 순서가 빠른 'a'를 return합니다.
- 세 번째 예시에서 "programming"은 'g'와 'm'이 2번으로 가장 많이 등장하므로 알파벳 순서가 빠른 'g'를 return합니다.
- 네 번째 예시에서 "aaa"는 'a'가 3번으로 가장 많이 등장합니다.
- 다섯 번째 예시에서 "abcde"는 모든 문자가 1번씩 등장하므로 알파벳 순서가 빠른 'a'를 return합니다.
